Cape Town Explorer

A modern travel companion for discovering Cape Town's culture, cuisine, and coastline.

Site Name Site Purpose Scenarios Color Schema Typography Wireframes

Site Name & Domain

Chosen Name

Cape Town Explorer — this name instantly communicates location and purpose. It positions the site as a curated travel companion for first-time visitors and locals exploring hidden gems.

Optional Domain

Availability checked: capetown-explorer.com (alternate: capetownxplorer.com)

Site Purpose

The Cape Town Explorer site will be a three-page responsive travel guide that helps users plan memorable itineraries. Core content pillars include:

  • Neighborhood Guides: highlight dining, culture, safety tips, and transport details.
  • Dynamic Attractions Library: filterable by mood (heritage, outdoors, foodie) with localStorage support for "saved" favorites.
  • Smart Planning Tools: weather snapshot, packing tips, and a booking-ready contact form for bespoke tours.

The final site will satisfy all WDD 131 project specs: valid HTML/CSS, mobile-first design, lazy-loaded media, JavaScript-driven interactivity, conditional logic, array methods, template literals, and localStorage persistence.

Target Audience & Scenarios

Primary audience: English-speaking travelers (ages 20-45) seeking curated, trustworthy recommendations with an emphasis on culture and outdoor adventures.

  1. "I only have 48 hours in Cape Town. What must-see experiences fit my schedule?"
  2. "Which neighborhoods are safe for solo travelers at night and how do I get around?"
  3. "Can I save my favorite attractions and revisit them later on any device?"

Typography

Playfair Display

Usage: Hero heading, section titles, CTA text. Selected for a sophisticated editorial feel aligned with premium travel brands.

Inter

Usage: Body copy, navigation, UI labels, form inputs. Chosen for clarity, excellent legibility, and broad language support.

This planning document already uses Playfair Display for headings and Inter for body text to demonstrate the typographic pairing.
